# 🧠 AI Research Assistant

An AI-powered multi-agent system that automates research by retrieving, summarizing, and analyzing academic papers.

---

## 🚀 Live web app

👉 https://ai-research-assistant-kqrsvh8tcairwilmqujem7.streamlit.app/

---

## ✨ Features

* 🤖 Multi-agent architecture (Researcher, Summarizer, Analyst, Coordinator)
* 📄 Automated research paper retrieval (ArXiv)
* 🧠 AI-powered summarization & insights (Gemini)
* 📊 Generates structured research reports
* 📄 Download report as PDF
* 🌐 Interactive Streamlit UI

---

## ⚙️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* ArXiv API
* ReportLab (PDF generation)

---

## 🧠 How It Works

1. Research Agent fetches papers
2. Summarizer Agent extracts key points
3. Analyst Agent finds insights & gaps
4. Coordinator compiles final report
